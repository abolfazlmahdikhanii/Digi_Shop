# Digi_Shop

This project is a demo of store sites with the difference that instead of using the database, local storage is used.

Project features:

- 1.search product
- 2.add to cart
- 3.register & login
- 4.add product
- 5.edit product
- 6.delete product
- 7.show last update
- and ...

[Demo Site](https://shoppin-digi.netlify.app)
